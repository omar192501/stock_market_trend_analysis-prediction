# 📊 Stock Market Trend Analysis & Prediction

## 📌 Overview
This project analyzes stock price behavior of Apple (AAPL) using historical data and machine learning techniques. It focuses on trend analysis, risk evaluation, and price prediction.

## 📊 Dataset
- Source: Yahoo Finance (yfinance)
- Period: 2021–2024

## 🧹 Data Processing
- Created moving averages (MA10, MA50)
- Calculated volatility
- Created prediction variable using shift(-1)

## 📈 Analysis
- Trend visualization
- Risk (volatility) analysis
- Moving average indicators

## 🤖 Model
- Random Forest Regressor
- Predicts next-day stock price

## 📊 Results
- Model captures general trends
- Less accurate during high volatility periods
